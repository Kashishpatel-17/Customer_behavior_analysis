# Customer_behavior_analysis
This project analyzes 3,900 customer purchases to uncover key insights into shopping behavior, spending patterns, customer segments, and product preferences. The findings help businesses optimize marketing strategies, boost subscriptions, enhance loyalty programs, and improve product positioning for better revenue growth.
